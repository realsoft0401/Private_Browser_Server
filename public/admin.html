运维边界

Demo 边界
普通 bind 前会检查 Client 本地 node-registration.json。 只要存在本地注册锁，普通 bind 必须拒绝。旧 Node 不可用时， 管理员手动删除 Client 本地 /Business/data/node-registration.json 后， 再由新 Node 发起普通 bind。当前不提供 force bind 接口。 Browser Env 的加载、卸载和备份只走 Node Server API，页面会先展示 run admission、slot 缓存和环境包状态， 禁止把“slot 容器是 running”误解成“环境包已经允许加载运行”。
